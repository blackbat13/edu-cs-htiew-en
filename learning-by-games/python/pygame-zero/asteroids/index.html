    Nasz cel¶

    Zaoenie gry jest proste: poruszamy statkiem u dou ekranu, omijamy asteroidy, strzelamy do nich i zdobywamy punkty! Powysza animacja pokazuje, jak bdzie wyglda nasza gra.

    Wstpna konfiguracja¶

    Zaczynamy standardowo: tworzymy nowy projekt, instalujemy bibliotek, pobieramy materiay i umieszczamy je w odpowiednich miejscach. Nasz projekt moemy nazwa np. "Asteroids" albo "Asteroidy". Gdy ju utworzymy projekt, tworzymy w nim nowe katalogi images, sounds oraz music. Nastpnie pobieramy wyej wymienione materiay, rozpakowujemy je, a zawarto przerzucamy do odpowiednich katalogw. Pozostao nam jeszcze zainstalowa bibliotek: w okienku terminala wypisujemy standardowo polecenie pip install pgzero.

    Gwnym aktorem naszej gry, w ktrego wcieli si gracz, bdzie statek kosmiczny. Bdziemy go wywietla na dole ekranu i porusza na boki za pomoc myszy.

    Poruszamy statkiem¶

    Statkiem bdziemy porusza w lewo/prawo. Nasza posta bdzie sterowana za pomoc myszki: statek bdzie lecia w kierunku, w ktrym znajduje si wskanik myszy. W tym celu musimy nada naszemu statkowi jak prdko poziom. Dopiszemy wic do naszego statku zmienn vx z pocztkow wartoci np. \(5\), zaraz pod ustaleniem pozycji statku na ekranie.

    ship.vx = 5

    W czci aktualizujcej usuwamy instrukcj pass. Na pocztku odczytamy aktualn pozycj wskanika myszy za pomoc funkcji pygame.mouse.get_pos() z biblioteki pygame. Poniewa funkcja ta zwraca nam wsprzdne wskanika myszy (\(x\) oraz \(y\)), to jej wynik zapiszemy w dwch zmiennych: mouse_x oraz mouse_y.

    def update():
    mouse_x, mouse_y = pygame.mouse.get_pos()

    Teraz pozostao nam sprawdzi, z ktrej strony znajduje si wskanik myszy wzgldem statku: z lewej czy z prawej. Jeeli wsprzdna \(x\) myszy jest mniejsza od wsprzdnej \(x\) statku, to znaczy, e mysz znajduje si z lewej strony statku. Dopisujemy wic instrukcj warunkow ze wspomnianym warunkiem na koniec funkcji update.

    def update():
    ...
    if mouse_x < ship.x:

    Jeeli tak jest, to powinnimy statek przesun w lewo zgodnie z jego prdkoci. W tym celu odejmujemy prdko statku (vx) od jego wsprzdnej \(x\).

    def update():
    ...
    if mouse_x < ship.x:
        ship.x -= ship.vx

    Podobnie postpujemy przy ruchu w praw stron. Zaczynamy od warunku: sprawdzamy, czy wsprzdna \(x\) myszy jest wiksza od wsprzdnej \(x\) statku.

    def update():
    ...
    if mouse_x > ship.x:

    Poruszamy statkiem w prawo, tzn. dodajemy jego prdko do pozycji \(x\).

    def update():
    ...
    if mouse_x > ship.x:
        ship.x += ship.vx

    Pena funkcja update bdzie wic wygldaa tak jak poniej.

    def update():
    mouse_x, mouse_y = pygame.mouse.get_pos()
    if mouse_x < ship.x:
        ship.x -= ship.vx
    if mouse_x > ship.x:
        ship.x += ship.vx

    Asteroidy¶

    Gwne zagroenie w naszej grze bd stanowi asteroidy "spadajce" z gry ekranu.

W tym celu sprawdzimy, czy wylosowana liczba rzeczywista z przedziau \(<0, 1)\) jest mniejsza od jakiej ustalonej wartoci, np. \(0.02\). Liczb rzeczywist wylosujemy za pomoc funkcji random.random().     Teraz czas na aktualizacj pozycji asteroid. W tym celu musimy przej przez wszystkie elementy naszej listy i dla kadego wykona odpowiednie operacje. Robimy to w czci aktualizujcej.

    Poniewa bdziemy usuwa asteroidy, ktre wyleciay poza ekran, to w ptli musimy przej przez kopi listy asteroid, ktr tworzymy piszc dwukropek w nawiasach kwadratowych po nazwie listy: asteroids_list[:].

    Dla zachowania porzdku aktualizacj asteroid zapiszemy w nowej funkcji update_asteroids, ktr utworzymy zaraz pod funkcj update.

    Zaczynamy od napisania ptli przechodzcej przez wszystkie asteroidy w kopii listy.

    for asteroid in asteroids_list[:]:

    Dla kadej asteroidy bdziemy przemieszcza j zgodnie z jej prdkoci, wic do jej wsprzdnej \(y\) dodajemy jej prdko vy.

    def update_asteroids():
    for asteroid in asteroids_list[:]:
        asteroid.y += asteroid.vy

    eby nasza gra nie spowalniaa po pewnym czasie, powinnimy na bieco usuwa asteroidy, ktrych ju nie wida na ekranie. Dlatego dopisujemy w ptli instrukcj warunkow sprawdzajc, czy asteroida wyleciaa poza ekran, tzn. czy jej wsprzdna \(y\) jest wiksza od \(HEIGHT + 50\) (dodajemy \(50\) tak by caa asteroida zdya wylecie poza ekran).

        asteroid.y += asteroid.vy
        if asteroid.y > HEIGHT + 50:

    Jeeli asteroida opucia ekran gry to usuwamy j z listy za pomoc metody remove, jako argument podajc asteroid do usunicia, podobnie jak robilimy przy dodawaniu asteroidy do listy.

    def update_asteroids():
    for asteroid in asteroids_list[:]:
        asteroid.y += asteroid.vy
        if asteroid.y > HEIGHT + 50:
            asteroids_list.remove(asteroid)

    Pozostao nam wywoa nasz now funkcj na kocu funkcji update.

import pygame
import random

HEIGHT = 900

ship = Actor("ship")
ship.x = WIDTH / 2
ship.y = HEIGHT - 60
ship.vx = 5

asteroids_list = []  # Lista, w ktrej zapisujemy wszystkie asteroidy

    for asteroid in asteroids_list:  # Przechodzimy ptl przez wszystkie asteroidy
        asteroid.draw()  # Rysujemy asteroid na ekranie gry

    if mouse_x < ship.x:
        ship.x -= ship.vx

    if mouse_x > ship.x:
        ship.x += ship.vx

    Niszczymy asteroidy¶

    C to za lasery, ktre nie niszcz asteroid? Mona powiedzie, e s to cakowicie normalne lasery, ale nie w naszej grze! Czas zniszczy kilka asteroid. W tym celu utworzymy now funkcj update_lasers_hits, ktr dopiszemy zaraz pod nasz poprzedni funkcj update_lasers.

    def update_lasers_hits():

    Dla kadego laseru bdziemy sprawdza, czy trafi on w ktr z asteroid. Jeeli tak, to usuniemy zarwno laser jak i asteroid. Dopisujemy wic ptl, ktra przejdzie przez wszystkie lasery w kopii listy lasers_list.

    def update_lasers_hits():
    for laser in lasers_list[:]:

    Asteroid mamy wiele, jak wic sprawdzi, czy w ktr trafi laser? Potrzebna nam kolejna ptla, ktra przejdzie przez wszystkie asteroidy w kopii listy asteroids_list.

    def update_lasers_hits():
    for laser in lasers_list[:]:
        for asteroid in asteroids_list[:]:

    Mamy ju zarwno laser jak i asteroid. Sprawdmy wic, czy s w kolizji! W tym celu skorzystamy ze znanej nam funkcji colliderect.

    def update_lasers_hits():
    for laser in lasers_list[:]:
        for asteroid in asteroids_list[:]:
            if laser.colliderect(asteroid):

    Jeeli laser trafi w asteroid, to usuwamy zarwno laser jak i asteroid z odpowiednich list za pomoc metody remove.

    def update_lasers_hits():
    for laser in lasers_list[:]:
        for asteroid in asteroids_list[:]:
            if laser.colliderect(asteroid):
                lasers_list.remove(laser)
                asteroids_list.remove(asteroid)

    Dopiszmy teraz wywoanie naszej nowej funkcji na koniec funkcji update.

    def update():
    ...
    update_lasers_hits()

    Gdybymy teraz przetestowali nasz gr, to moe si zdarzy, e czasem wyczy si z komunikatem bdu w konsoli. Dzieje si tak, gdy laser trafi w jedn asteroid, usuniemy go z listy, ale jest take jednoczenie w kolizji z jeszcze inn asteroid. Wwczas sprbujemy usun ten sam laser drugi raz, ale przecie ju go wczeniej usunlimy, wic prba usunicia usunitego lasera zakoczy si bdem. Moemy jednak temu zapobiec w bardzo prosty sposb. Wystarczy e dopiszemy do naszej funkcji update_lasers_hits instrukcj break zaraz po usuniciu lasera i asteroidy. Instrukcja break sprawi, e wyjdziemy z ptli przechodzcej po wszystkich asteroidach i przejdziemy do kolejnego lasera.

    for laser in lasers_list[:]:
        for asteroid in asteroids_list[:]:
            if laser.colliderect(asteroid):
                lasers_list.remove(laser)
                asteroids_list.remove(asteroid)
                break

    Kocowe poprawki¶

    Na koniec zrobimy jeszcze kilka rzeczy: ukryjemy wskanik myszy, odtworzymy muzyk w tle i wybierzemy losow grafik dla asteroid.

    Ukrywamy wskanik myszy¶

    Ukrycie wskanika myszy zapiszemy na samym kocu naszego kodu, zaraz przed instrukcj pgzrun.go(). Aby ukry wskanik skorzystamy z metody pygame.mouse.set_visible z biblioteki Pygame. Jako parametr, w okrgych nawiasach, podamy warto False, ktra oznacza, e wskanik myszy nie ma by widoczny.

    pygame.mouse.set_visible(False)

    Odtwarzamy muzyk¶

    Odtworzenie muzyki rwnie zapiszemy na kocu kody, zaraz przed instrukcj pgzrun.go(). Zrobimy to korzystajc z metody music.play, jako parametr podajc nazw pliku dwikowego z katalogu music. Nasza muzyka bdzie odtwarzana w ptli przez cay czas trwania rozgrywki.

    music.play("space")

    Poniewa muzyka moe by do gona, to warto ustawi jej gono na warto \(0.3\), korzystajc zmetody music.set_volume.

    music.play("space")
music.set_volume(0.3)

    Losowa grafika dla asteroid¶

    Gdy przyjrzymy si naszym grafikom to zobaczymy, e mamy kilka grafik przedstawiajcych asteroidy. Grafiki ponumerowane s od \(1\) do \(4\). My natomiast uywamy tylko pierwszej grafiki. CZas to zmieni! W tym celu zmodyfikujemy nasz funkcj add_asteroid dodajc now asteroid do gry. Na samym jej pocztku wylosujemy losow liczb z przedziau od \(1\) do \(4\) za pomoc metody random.randint i zapiszemy j w zmiennej image_id.

    def add_asteroid():
    image_id = random.randint(1, 4)
    ...

    Teraz pozostao nam zmodyfikowa linijk tworzc nowego aktora reprezentujcego asteroid. Z nazwy grafiki usuniemy liczb \(1\) pozostawiajc sam wyraz asteroid, do ktrego dopiszemy nasz liczb image_id zamienion na tekst (str(image_id)).

    def add_asteroid():
    image_id = random.randint(1, 4)
    asteroid = Actor("asteroid" + str(image_id))

    import pgzrun
import pygame
import random # Biblioteka do liczb losowych

WIDTH = 600  # Ustawiamy szeroko okna gry
HEIGHT = 900  # Ustawiamy wysoko okna gry

ship = Actor("ship")  # Tworzymy nowego aktora reprezentujcego statek
ship.x = WIDTH / 2  # Ustawiamy pozycj poziom statku na poow szerokoci ekranu
ship.y = HEIGHT - 60  # Ustawiamy statek na dole ekranu, z niewielkim marginesem
ship.vx = 5  # Zapamitujemy prdko poziom statku
ship.points = 0  # Zapamitujemy liczb zdobytych punktw
ship.lifes = 3  # Zapamitujemy liczb dostpnych y
ship.ammunition = 5  # Liczba dostpnej amunicji

asteroids_list = []  # Lista, w ktrej zapisujemy wszystkie asteroidy
lasers_list = []  # Lista, w ktrej zapisujemy wszystkie lasery

def draw():  # Funkcja rysujca elementy gry
    screen.blit("bg", (0, 0))  # Rysujemy to gry
    for asteroid in asteroids_list:  # Przechodzimy ptl przez wszystkie asteroidy
        asteroid.draw()  # Rysujemy asteroid na ekranie gry

    for laser in lasers_list:  # Przechodzimy przez wszystkie lasery na licie
        laser.draw()  # Rysujemy laser na ekranie gry

    ship.draw()  # Rysujemy statek na ekranie
    screen.draw.text(str(ship.points), center=(WIDTH / 2, 20), fontsize=50, color="yellow")  # Wywietlamy liczb zdoboytych punktw na ekranie gry
    draw_lifes()  # Rysujemy ycia na ekranie
    if ship.lifes <= 0:  # Jeeli skoczyy nam si ycia
        screen.draw.text("GAME OVER", center=(WIDTH / 2, HEIGHT / 2), fontsize=90, color="yellow")  # Wywietlamy napis "GAME OVER" na rodku ekranu gry

def draw_lifes():  # Pomocnicza funkcja rysujca ycia
    for life_id in range(1, ship.lifes + 1):  # Przechodzimy ptl od jeden do liczby dostpnych y wcznie
        life = Actor("life")  # Tworzymy aktora reprezentujcego ycie
        life.x = life_id * life.width  # Ustawiamy pozycj poziom aktora
        life.y = life.height / 2  # Ustawiamy pozycj aktora w pionie
        life.draw()  # Rysujemy ycie na ekranie

def update():  # Funkcja aktualizujca elementy gry
    if ship.lifes <= 0:  # Jeeli skoczyy nam si ycia
        return  # Koczymy dziaanie funkcji

    mouse_x, mouse_y = pygame.mouse.get_pos()  # Odczytujemy obecn pozycj wskanika myszy
    if mouse_x < ship.x:  # Jeeli wskanik myszy jest z lewej strony statku
        ship.x -= ship.vx  # Przemieszczamy statek w lewo zgodnie z jego prdkoci

    if mouse_x > ship.x:  # Jeeli wskanik myszy jest z prawej strony statku
        ship.x += ship.vx  # Przemieszczamy statek w prawo zgodnie z jego prdkoci

    if random.random() < 0.02:  # Jeeli wylosowalimy odpowiednio ma liczb
        add_asteroid()  # Dodajemy now asteroid do gry

    update_asteroids()  # Aktualizujemy pozycje asteroid
    update_lasers()  # Aktualizujemy pozycje laserw
    update_lasers_hits()  # Aktualizujemy trafienia laserw w asteroidy

def update_asteroids():  # Pomocnicza funkcja aktualizujca pozycj asteroid
    for asteroid in asteroids_list[:]:  # Przechodzimy przez kopi listy asteroid
        asteroid.y += asteroid.vy  # Przemieszczamy asteroid zgodnie z jej prdkoci
        if asteroid.y > HEIGHT + 50:   # Jeeli asteroida wyleciaa poza ekran
            asteroids_list.remove(asteroid)  # Usuwamy j z listy asteroid
        elif asteroid.colliderect(ship):  # Sprawdzamy, czy asteroida zderzya si ze statkiem
            ship.lifes -= 1  # Zmniejszamy liczb pozostaych y
            asteroids_list.remove(asteroid)  # Usuwamy asteroid z gry
            if ship.lifes > 0:  # Jeeli pozostay jeszcze jakie ycia
                sounds.shield.play()  # Odtwarzamy dwik tarczy
            else:  # W przeciwnym przypadku, gdy ycia si ju skoczyy
                sounds.explosion.play()  # Odtwarzamy dwik wybuchu

def update_lasers():  # Pomocnicza funkcja aktualizujca pozycje laserw
    for laser in lasers_list[:]:  # Przechodzimy przez wszystkie lasery na licie
        laser.y += laser.vy  # Przemieszczamy laser zgodnie z jego prdkoci
        if laser.y < -laser.height:  # Jeeli laser wylecia poza ekran
            lasers_list.remove(laser)  # Usuwamy laser z gry

def update_lasers_hits():  # Pomocnicza funkcja sprawdzajca uderzenia laserw w asteroidy
    for laser in lasers_list[:]:  # Przechodzimy przez wszystkie lasery na licie
        for asteroid in asteroids_list[:]:  # Przechodzimy przez wszystkie asteroidy na licie
            if laser.colliderect(asteroid):  # Sprawdzamy, czy laser trafi w asteroid
                lasers_list.remove(laser)  # Usuwamy laser z gry
                asteroids_list.remove(asteroid)  # Usuwamy asteroid z gry
                ship.points += 1  # Zwikszamy liczb punktw po zestrzeleniu asteroidy
                break  # Koczymy ptl przechodzc po asteroidach i przechodzimy do kolejnego lasera

def add_asteroid():   # Pomocnicza funkcja dodajca now asteroid do gry
    image_id = random.randint(1, 4)  # Losujemy numer grafiki asteroidy
    asteroid = Actor("asteroid" + str(image_id))  # Tworzymy nowego aktora na podstawie losowej grafiki asteroidy
