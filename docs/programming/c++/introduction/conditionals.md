# Instrukcja warunkowa

## Instrukcja prosta

```cpp
if (warunek) {
    operacje;
}
```

## Instrukcja pełna

```cpp
if (warunek) {
    operacje;
} else {
    operacje;
}
```

## Instrukcja złożona

```cpp
if (warunek) {
    operacje;
} else if (warunek) {
    operacje;
} else {
    operacje;
}
```
