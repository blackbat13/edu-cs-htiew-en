# Solution 5

## Treść zadania

Napisz program zgodny z poniższą specyfikacją.

### Specification

#### Input

* $a$ - liczba naturalna

#### Output

* Pierwiastek z $a$

## Solution

```python
import math

a = int(input("Podaj liczbę naturalną: "))

pierwiastek = math.sqrt(a)

print(f"Pierwiastek z {a} = {pierwiastek}")
```
