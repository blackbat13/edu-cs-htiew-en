# Solution 2

## Treść zadania

Napisz program zgodny z poniższą specyfikacją.

### Specification

#### Input

* $a, b$ - dwie liczby całkowite

#### Output

* Suma liczb $a$ i $b$ 

## Solution

```python
a = int(input("Podaj pierwszą liczbę: "))
b = int(input("Podaj drugą liczbę: "))
   
suma = a + b

print(f"{a} + {b} = {suma}")
```
