# Ćwiczenie 3

Zapoznaj się z poniższą specyfikacją oraz pseudokodem, a następnie rozwiąż zadania.

## Specyfikacja

### Dane

* $x, y$ - liczby naturalne.

## Pseudokod

```
1. wx := x
2. wy := y
3. Dopóki wx != wy, wykonuj:
    4. Jeżeli wx > wy, to: 
        5. wy := wy + y
    6. W przeciwnym przypadku:
        7. wx := wx + x
        
8. Wypisz wx
```
