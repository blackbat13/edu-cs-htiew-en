# Zadanie 1 - rozwiązanie

## Symulacja

```
[x = 10, y = 3]

1. [q = 0]
2. [r = 10]

3. Dopóki 3 <= 10 - OK
    1. [q = 0 + 1 = 1]
    2. [r = 10 - 3 = 7]
    
4. Dopóki 3 <= 7 - OK
    1. [q = 1 + 1 = 2]
    2. [r = 7 - 3 = 4]
    
5. Dopóki 3 <= 4 - OK
    1. [q = 2 + 1 = 3]
    2. [r = 4 - 3 = 1]
    
6. Dopóki 3 <= 1 - NIE (koniec pętli)

7. Wypisz 3
8. Wypisz 1    
```

## Wynik

$q=3,\ r=1$
