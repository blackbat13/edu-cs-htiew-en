# Ćwiczenie 1

Zapoznaj się z poniższą specyfikacją oraz pseudokodem, a następnie rozwiąż zadania.

## Specyfikacja

### Dane

* $x$, $y$ - liczby naturalne, $x > 0$, $y > 0$. 

## Pseudokod

```
1. q := 0
2. r := x
3. Dopóki y <= r, wykonuj:
    4. q := q + 1
    5. r := r - y
    
6. Wypisz q
7. Wypisz r 
```

## Zadanie 1

Podaj wynik działania programu dla danych: `x := 10, y := 3`
